Perguntas Frequentes (FAQ)

Como abrir uma nova aba no navegador?

Para abrir uma nova aba no navegador:

  1. Localize o botão "+" ao lado das abas abertas
  2. Clique neste botão
  3. Você também pode usar o atalho de teclado: Ctrl+T (Windows) ou Command+T (Mac)
Onde coloco o endereço do e-mail quando estou enviando uma mensagem?

Quando estiver escrevendo um novo e-mail:

  1. Procure o campo chamado "Para"
  2. Digite o endereço completo da pessoa (exemplo: pessoa@provedor.com)
  3. Certifique-se de digitar corretamente, incluindo o "@" e o domínio (.com, .com.br, etc)
Como salvar uma foto que recebi por mensagem?

Para salvar uma foto recebida:

  1. Toque na foto para abri-la em tela cheia
  2. Procure o ícone de download (geralmente uma seta para baixo)
  3. Toque neste ícone para salvar a foto no seu dispositivo
  4. As fotos salvas geralmente ficam na pasta "Downloads" ou "Imagens"
